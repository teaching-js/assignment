# How to get started

## Backend

You can use virtual env [recommend]. Make sure you're using python3.

```bash
cd backend
virtualenv env
source env/bin/activate
pip install -r requirements.txt
python app.py
```

or if you don't mind doing things globally:

```
cd backend
pip3 install -r requirements.txt
python3 app.py
```

## Frontend

```
cd frontend
npm install
npm start
```

See Frontend README.
